# Code of Conduct

Realm is a technical project. Be direct, respectful, and specific.

## Expected Behavior

- Keep discussion focused on reliability, usability, documentation, security,
  and maintainability.
- Critique code, behavior, and tradeoffs instead of people.
- Provide reproduction steps, screenshots, logs, or concrete examples when
  reporting problems.
- Respect security boundaries: do not post real secrets, private project files,
  or exploit details that should go through the security process.
- Assume contributors use different operating systems, languages, terminals,
  and model-provider setups.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private credentials, private project data, or personal information.
- Spamming issues, pull requests, or discussions with unrelated promotion.
- Intentionally disruptive behavior that blocks maintainers from triaging or
  shipping the project.

## Enforcement

Maintainers may edit or hide comments, close issues, reject pull requests, or
block accounts when behavior makes useful technical collaboration impossible.
Security-sensitive reports may be redirected to GitHub Security Advisories.
